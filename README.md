# CodeLabAPIEmail

API de Email CodeLab

## Development Start

```bash
docker compose up --build
```

## Excecução de testes

- Atenção: "attach" ao container para executar os testes.
- Caso não queira o coverage, remova o cov

## Unit

```bash
npm run test:cov
```

## E2E

```bash
npm run test:e2e:cov
```
